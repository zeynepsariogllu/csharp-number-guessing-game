# csharp-number-guessing-game

## Overview

This project is a Windows Forms number guessing game developed using C# and .NET.

The player attempts to guess a randomly generated 4-digit number with unique digits. After each guess, the application provides feedback about how many digits are in the correct position and how many digits exist in the target number but are placed incorrectly.

## Features

* Random 4-digit number generation
* Unique digit validation
* Input validation
* Score tracking system
* Maximum attempt limit
* Interactive game feedback
* Guess history display
* Windows Forms graphical interface

## Game Rules

* The target number consists of 4 unique digits.
* The number cannot start with zero.
* Players have 5 attempts to find the correct number.
* Each incorrect guess reduces the score by 50 points.
* The game starts with 500 points.
* Feedback is provided after every guess.
